# periodicity

## Описание
Свойство `periodicity` определяет периодичность машины.

### Пример:
``` javascript
motor.periodicity = 100;
```

## Принимаемые значения
- `Motor.SR` (0): stator - rotor

## Тип значения свойства
`Number`

## Доступ
`Только Чтение`
