# Airgap.ID

## Описание
Идентификатор воздушного зазора.

## Значение
 - Airgap.Empty : 0
 - Airgap.ID1 : 1
 - Airgap.ID2 : 2
 - Airgap.ID3 : 3
 - Airgap.ID4 : 4

## Пример
``` javascript linenums="1"
let id = Airgap.ID1
console.info(id) // OUTPUT: 1
```
