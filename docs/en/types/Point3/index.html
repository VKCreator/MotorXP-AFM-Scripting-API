    Описание типа Point3¶

    Встроенный тип Point3 представляет точку в 3D пространстве.

    Свойства типа Point3¶

    Ниже приведен список доступных свойств, предоставляемых типом Point3. Щелкните имя свойства, чтобы увидеть его подробное описание и примеры.

    Свойство Описание
    x Координата точки по оси X.
    y Координата точки по оси Y.
    z Координата точки по оси Z.

    Методы типа Point3¶

    Ниже приведен список доступных методов, предоставляемых типом Point3. Щелкните имя метода, чтобы увидеть его подробное описание и примеры.

    Геометрические преобразования точки¶

    Отражение¶

    Метод Описание
    mirrorO() Отражение относительно начала координат.
    mirrorX() Отражение относительно оси X.
    mirrorY() Отражение относительно оси Y.
    mirrorZ() Отражение относительно оси Z.
    mirrorXY() Отражение относительно плоскости XY.
    mirrorXZ() Отражение относительно плоскости XZ.
    mirrorYZ() Отражение относительно плоскости YZ.

    Перемещение¶

    Метод Описание
    move() Перемещает точку на заданный вектор.
    moveX() Перемещает точку вдоль оси X.
    moveY() Перемещает точку вдоль оси Y.
    moveZ() Перемещает точку вдоль оси Z.

    Поворот¶

    Метод Описание
    rotate() Поворачивает точку вокруг произвольной оси.
    rotateX() Поворачивает точку вокруг оси X.
    rotateY() Поворачивает точку вокруг оси Y.
    rotateZ() Поворачивает точку вокруг оси Z.

    Масштабирование¶

    Метод Описание
    scale() Масштабирует точку по всем осям.
    scaleX() Масштабирует точку по оси X.
    scaleY() Масштабирует точку по оси Y.
    scaleZ() Масштабирует точку по оси Z.
    scaleXY() Масштабирует точку в плоскости XY.
    scaleYZ() Масштабирует точку в плоскости YZ.
    scaleXZ() Масштабирует точку в плоскости XZ.
    scaleXYZ() Масштабирует точку равномерно по всем осям.

    Трансляция¶

    Метод Описание
    translate() Транслирует точку на заданный вектор.
    translateX() Транслирует точку вдоль оси X.
    translateY() Транслирует точку вдоль оси Y.
    translateZ() Транслирует точку вдоль оси Z.

    Дополнительные методы типа¶

    Метод Описание
    distance() Расстояние между двумя точками.
