# isMiddle()

## Описание
Истина, если в моторе элемент статора средний.

## Синтаксис
```javascript
isMiddle() : bool
```

## Возвращаемое значение
`bool`: Истина, если в моторе элемент статора средний.

## Пример
```javascript linenums="1"
let result = motor.stator.item(StatorItem.ID1).isMiddle()
console.info(result)
```
