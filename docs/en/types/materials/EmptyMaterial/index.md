# Описание `EmptyMaterial`
Материал пустой, предназначен для задания материала деталей мотора `Piece`.

## Синтаксис
```javascript
let m1 = Material.empty() // Способ 1
let m2 = new EmptyMaterial() // Способ 2
```

## Свойства `EmptyMaterial`

Нет свойств.

## Методы `EmptyMaterial`

Нет методов.
