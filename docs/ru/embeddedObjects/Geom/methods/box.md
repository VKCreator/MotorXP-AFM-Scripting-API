# box()

## Описание
Описание метода.

## Синтаксис
```javascript
box(argument: type) : returnType
```

## Аргументы
- `argument` (type, обязательно): описание аргумента.

## Возвращаемое значение
`returnType`: описание возвращаемого значения.

## Пример
```javascript linenums="1"
let result = box(value);
console.info(result);
```
