# createQWidget()

## Описание
Создать виджет QWidget

## Синтаксис
```javascript
createQWidget(parent: QWidget) : QWidget
``` 

## Аргументы
- `parent` (QWidget, обязательно): Родительский виджет.

## Возвращаемое значение
- [`QWidget`]()

## Пример
``` javascript linenums="1"
let w = QtWidgets.createQWidget(parent)
```
