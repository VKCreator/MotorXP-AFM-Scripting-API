# Глобальные функции

## include(path)

### Description
Включение скрипта

### Syntax
``` javascript
include(path);
``` 

### Example
``` javascript linenums="1"
include('path_to_script');
``` 
## require(path)

### Description
Загрузка модуля

### Syntax
``` javascript
require(path);
``` 

### Example
``` javascript linenums="1"
let m = require(path)
```
