# empty()

## Описание
Пустой материал.

## Синтаксис
```javascript
Material.empty(): EmptyMaterial
``` 

## Возвращаемое значение
[`EmptyMaterial`](./../../../types/materials/EmptyMaterial/_index.md)

## Пример
``` javascript linenums="1"
var m = Material.empty();
console.info(m) // output: Not assigned
```
