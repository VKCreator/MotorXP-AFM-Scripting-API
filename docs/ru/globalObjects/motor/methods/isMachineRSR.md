# isMachineRSR()

## Описание
Истина, если тип машины `RSR`.

## Синтаксис
```javascript
motor.isMachineRSR() -> Boolean
```

## Возвращаемое значение
`#!javascript Boolean`: Истина, если тип машины `RSR`.

## Пример
```javascript linenums="1"
let result = motor.isMachineRSR()
console.info(result)
```
