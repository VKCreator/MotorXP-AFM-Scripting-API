# motor.isMachineRSRSR()

## Описание
Истина, если тип машины `RSRSR`.

## Синтаксис
```javascript
motor.isMachineRSRSR() -> Boolean
```

## Возвращаемое значение
| Тип     | Описание                                                                 |
|---------|--------------------------------------------------------------------------|
| `#!javascript Boolean`  | Истина, если тип машины `RSRSR`. |

## Пример
```javascript linenums="1"
let result = motor.isMachineRSRSR()
console.info(result)
```
