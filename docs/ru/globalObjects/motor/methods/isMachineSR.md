# isMachineSR()

## Описание
Истина, если тип машины `SR`.

## Синтаксис
```javascript
motor.isMachineSR() -> Boolean
```

## Возвращаемое значение
`#!javascript Boolean`: Истина, если тип машины `SR`.

## Пример
```javascript linenums="1"
let result = motor.isMachineSR()
console.info(result)
```
