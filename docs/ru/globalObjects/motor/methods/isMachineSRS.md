# motor.isMachineSRS()

## Описание
Истина, если тип машины `SRS`.

## Синтаксис
```javascript
motor.isMachineSRS() -> Boolean
```

## Возвращаемое значение
| Тип     | Описание                                                                 |
|---------|--------------------------------------------------------------------------|
| `#!javascript Boolean`  | Истина, если тип машины `SRS`. |

## Пример
```javascript linenums="1"
let result = motor.isMachineSRS()
console.info(result)
```
