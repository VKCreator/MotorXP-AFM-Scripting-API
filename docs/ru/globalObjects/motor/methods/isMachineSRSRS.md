# isMachineSRSRS()

## Описание
Истина, если тип машины `SRSRS`.

## Синтаксис
```javascript
motor.isMachineSRSRS() -> Boolean
```

## Возвращаемое значение
`#!javascript Boolean`: Истина, если тип машины `SRSRS`.

## Пример
```javascript linenums="1"
let result = motor.isMachineSRSRS()
console.info(result)
```
