# id

## Описание
Идентификатор воздушного зазора (Airgap.ID).

## Тип значения свойства
`#!javascript Number`

## Возможные значения
- Airgap.ID1
- Airgap.ID2
- Airgap.ID3
- Airgap.ID4

## Доступ
`Только Чтение`

## Пример
```javascript linenums="1"
let airgap = motor.airgap
let result = airgap.id
console.info(result) // output: 1
```
