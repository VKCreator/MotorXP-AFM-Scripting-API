# xCenter

## Описание
Середина по оси X

## Тип значения свойства:
`Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
xCenter
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let cx = bbox.xCenter
console.info(cx) // output:
```
