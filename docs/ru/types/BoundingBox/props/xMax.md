# xMax

## Описание типа
Максимальное значение по оси X.

## Тип значения свойства
`#!javascript Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
xMax
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let mx = bbox.xMax
console.info(mx) // output:
```
