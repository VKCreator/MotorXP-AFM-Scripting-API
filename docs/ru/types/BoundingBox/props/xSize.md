# xSize

## Описание
Размер по оси X.

## Тип значения свойства
`#!javascript Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
xSize
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let sx = bbox.xSize
console.info(sx) // output:
```
