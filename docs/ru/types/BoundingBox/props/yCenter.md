# yCenter

## Описание
Середина по оси Y

## Тип значения свойства:
`Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
yCenter
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let cy = bbox.yCenter
console.info(cy) // output:
```
