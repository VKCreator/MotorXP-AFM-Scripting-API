# yMin

## Описание
Минимальное значение по оси Y.

## Тип значения свойства
`#!javascript Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
yMin
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let my = bbox.yMin
console.info(my) // output:
```
