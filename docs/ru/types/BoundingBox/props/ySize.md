# ySize

## Описание
Размер по оси Y.

## Тип значения свойства:
`Number`

## Доступ
`Только Чтение`

## Синтаксис
``` javascript
ySize
```
## Пример
``` javascript linenums="1"
let cyl = Geom.cylinder(100, 50)
let bbox = cyl.boundBox()
let sy = bbox.ySize
console.info(sy) // output:
```
