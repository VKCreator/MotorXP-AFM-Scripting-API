# Rotor.MiddleType

## Описание
Определяет тип средней части ротора: с ярмом (`Yoke`) или без ярма (`Yokeless`).

## Значение
 - `Rotor.Yoke`: Средняя часть ротора с ярмом.
 - `Rotor.Yokeless`: Средняя часть ротора без ярма.

## Пример
```javascript linenums="1"
let value = Rotor.Yoke;
console.info(value);
```
