# countItems

## Описание
Количество элементов статора, зависит от motor.machineType.

## Тип значения свойства
`Number`

## Доступ
`Только Чтение`

## Синтаксис
```javascript
countItems
```

## Пример
```javascript linenums="1"
let result = motor.stator.countItems
console.info(result)
```
