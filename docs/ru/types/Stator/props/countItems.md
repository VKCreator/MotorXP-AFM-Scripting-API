# countItems

## Описание
Количество элементов статора, зависит от motor.machineType.

## Тип значения свойства
`number`

## Доступ
`Только Чтение`

## Синтаксис
```javascript
countItems
```

## Пример
```javascript linenums="1"
let result = motor.countItems
console.info(result)
```
