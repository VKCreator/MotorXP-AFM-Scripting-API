# numberSlots

## Описание
Количество пазов.

## Принимаемые значения:
Число из ряда значений: 3,6,9,12,15,...501.

## Тип значения свойства
`#!javascript Number`

## Доступ
`Чтение\Запись`

## Синтаксис
```javascript
numberSlots = value
```

## Пример
```javascript linenums="1"
let result = motor.stator.numberSlots
console.info(result)
```
