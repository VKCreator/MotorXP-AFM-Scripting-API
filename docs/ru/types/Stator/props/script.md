# script()

## Описание
Описание метода.

## Синтаксис
```javascript
script(argument: type) : returnType
```

## Аргументы
- `argument` (type, обязательно): описание аргумента.

## Возвращаемое значение
`returnType`: описание возвращаемого значения.

## Пример
```javascript linenums="1"
let result = script(value);
console.info(result);
```
