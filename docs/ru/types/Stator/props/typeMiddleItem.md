# typeMiddleItem

## Описание
Тип средней части статора

## Принимаемые значения:
Одно из зачений:
 - `Stator.Yoke`
 - `Stator.Yokeless`

## Тип значения свойства
`Stator.MiddleType`

## Доступ
`Чтение\Запись`

```javascript
typeMiddleItem = value
```
