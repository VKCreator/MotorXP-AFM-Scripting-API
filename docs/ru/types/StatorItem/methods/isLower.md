# isLower()

## Описание
Истина, если в моторе элемент статора нижний.

## Синтаксис
```javascript
isLower() -> Boolean
```

## Возвращаемое значение
`Boolean`: Истина, если в моторе элемент статора нижний.

## Пример
```javascript linenums="1"
let result = motor.stator.item(StatorItem.ID1).isLower()
console.info(result)
```
