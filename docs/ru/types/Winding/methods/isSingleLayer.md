# isSingleLayer()

## Описание
Истина, если задан одинарный слой.

## Синтаксис
```javascript
isSingleLayer() -> Boolean
```

## Возвращаемое значение
`#!javascript Boolean`: Истина, если задан одинарный слой.

## Пример
```javascript linenums="1"
let result = motor.winding.isSingleLayer()
console.info(result)
```
