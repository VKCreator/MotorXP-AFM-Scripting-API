# isWindingModelFull()

## Описание
Истина, если задана модель `Full`.

## Синтаксис
```javascript
isWindingModelFull() -> Boolean
```

## Возвращаемое значение
`Boolean`: Истина, если задана модель `Full`.

## Пример
```javascript linenums="1"
let result = motor.winding.isWindingModelFull()
console.info(result)
```
