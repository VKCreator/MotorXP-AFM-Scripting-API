# Описание типа `CustomMaterial`
Материал, задаваемый пользователем. Предназначен для задания материала деталям `Piece` мотора.

## Синтаксис
```javascript
let m1 = Material.custom(Qt.red) // Способ 1
let m2 = new CustomMaterial(Qt.red) // Способ 2
```
