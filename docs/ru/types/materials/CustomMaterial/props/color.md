# color

## Описание
Цвет материала.

## Принимаемые значения:
Одно из значений [^^`Qt.Colors`^^](./../../../../constants/Colors.md)

## Тип значения свойства
`QColor`

## Доступ
`Чтение\Запись`

## Синтаксис
```javascript
color = value
```

## Пример
```javascript linenums="1"
let m = Material.custom(Qt.red)
m.color = Qt.green
let result = m.color
console.info(result)
```
