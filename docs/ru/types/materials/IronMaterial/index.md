# Описание `IronMaterial`
Материал для сердечника, предназначен для задания материала деталей мотора `Piece`.

## Синтаксис
```javascript
let m1 = Material.iron() // Способ 1
let m2 = new IronMaterial() // Способ 2
```

## Свойства `IronMaterial`

Нет свойств.

## Методы `IronMaterial`

Нет методов.
